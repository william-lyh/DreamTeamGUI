#Dream Team GUI

1. Verify that Python version >=3.8
    ```
    python3 -V
    >>> Python 3.8
    ```
2. Initialize virtual environment with:
    ```
    python3 -m venv venv
    ```
3. Activate virtual environment:
    ```
    source .venv/bin/activate
    ```
4. Install required packages:
    ```
    pip3 install -r requirements.txt
    ```
5. Run flask server
    ```
    python3 app.py
    ```
